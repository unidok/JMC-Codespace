# Информация
Вся актуальная информация на Modrinth:
- https://modrinth.com/mod/jmc-codespace


## Этот мод помогает кодить на JustMC Creative+:
- Позволяет очищать код
- Позволяет сохранять код в файл
- Позволяет получить ссылку на загрузку кода
- Позволяет загрузить файл с кодом в мир
- Добавляет навигацию по коду при помощи Shift + ЛКМ
- Позволяет искать блоки в коде
- Позволяет получить код шаблона

## Использование
- `/csp clear` - очистить код
- `/csp config open` - открыть папку с конфигом
- `/csp config reload` - перезагрузить конфиг
- `/csp config reset` - сбросить конфиг
- `/csp config savingPeriod` - узнать текущий период сохранения шаблонов (тики)
- `/csp config savingPeriod <число>` - установить период сохранения шаблонов (тики)
- `/csp save stop` - остановить сохранение кода
- `/csp save file` - сохранить код в файл
- `/csp save file <имя>` - сохранить код в файл с указанным именем
- `/csp save upload` - получить ссылку на загрузку кода
- `/csp save fast file` - сохранить код в файл (быстрый режим)
- `/csp save fast file <имя>` - сохранить код в файл с указанным именем (быстрый режим)
- `/csp save fast upload` - получить ссылку на загрузку кода (быстрый режим)
- `/csp modules delete <файл>` - удалить сохранённый код в указанном файле
- `/csp modules clear` - удалить все сохранённые коды
- `/csp modules optimize <файл>` - оптимизировать код модуля
- `/csp modules load <файл>` - загрузить код из файла в мир
- `/csp modules load force <файл>` - загрузить код из файла в мир с заменой текущего кода
- `/csp search <ввод>` - найти в коде блоки по имени
- `/csp search-page <страница>` - показать определённую страницу результатов поиска
- `/csp template get-raw` - получить код шаблона в сжатом виде
- `/csp template get-json` - получить код шаблона в виде JSON
- `/csp template optimize` - оптимизировать код шаблона
